Замечания:

MISC
1. На двунаправленной шине данных, подключенной к STM стоит только 5-и вольтовая логика, т.к. все входы МК совместимы с 5В.
2. На входные сигналы идущие в ПЛИС стоит 3-х вольтовая логика тоже совместимую с 5В.
3. На BUSR3 вместо диода стоит транзистор.
4. Разъемы для RJ-45 выбраны 20см с контактом на корпус.

MCU
1. Поставлен резистор 220 Ом последовательно с кварцем. В части тестовых плат он есть, на части нет.
2. BOOT0 = "0", загрузка из FLASH. BOOT1 - не используется.
3. PDR_ON = VDD, внутренняя схема сброса включена.
4. VCAP_1(2) - 2.2 мкФ.
5. 4-е конденсатора 0805 2.2 мкФ, вместо двух по 4.7 мкФ. 12-ть конденсаторов 0805 по 0.1 мкФ. Количество и размеры конденсаторов может измениться по ходу разводки. 
6. VBAT, VUSB напрямую к VDD. 
7. VDDA через бусину к VDD, а так же конденсаторы 1.0 и 0.1 мкФ.
8. VREF+ напрямую к VDDA, а также конденсаторы 1.0 и 0.1 мкФ).
9. Кварц подключен через 220 Ом. На части демоплат этот резистор есть, на части нет. Размер 0805, на демо - скорее всего 0603.
10. Выводы JTAG имеют внутреннюю подтяжку и нет необходимости использовать внешние элементы.

E1
1. Последовательно на выходы/выходы E1 стоят нулевые резисторы 0805, чтобы при необходимости заменить на другой номинал.
2. JTAG TRST подключен к земле "For normal signal processing, this pin should be connected to ground".
3. REF подключен к земле через 0805 3кОм 1%. 
4. IC_7, IC_39 посажены на землю, IC_42 в воздухе.

CPLD
1. Ножки пока назначены для вида. Перестановка будет при разводке.
2. Добавил сигнал #DEV_CLR идущий с одной из ножек CPLD на ножку DEV_CLRn как на БЦК.
